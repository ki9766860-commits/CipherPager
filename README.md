# 🔐 CipherPager

نظام متكامل لتشفير وفك تشفير النصوص: تشفير البيجر (أرقام) + شيفرة مورس

## 🚀 تشغيل المشروع
فقط افتح `index.html` في المتصفح

## 📟 نظام البيجر
- تشفير: نص عربي → أرقام (مثال: سلام → 13-23-1-13)
- فك: أرقام → نص عربي

## 📡 شيفرة مورس
- تشفير: نص → مورس
- فك: مورس → نص
(يدعم عربي، إنجليزي، أرقام)

## 🛠️ التقنيات
HTML5 + CSS3 + JavaScript
