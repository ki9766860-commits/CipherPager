# 🔐 CipherPager

**CipherPager** هو تطبيق ويب تفاعلي يهدف إلى توفير بيئة آمنة وسهلة الاستخدام لتشفير وفك تشفير النصوص باستخدام تقنيتين كلاسيكيتين:

1. **نظام تشفير البيجر (Pager Encryption)** – تحويل الحروف العربية إلى أرقام (1-28) والعكس، مستوحى من أنظمة النداء الآلي القديمة.
2. **شيفرة مورس (Morse Code)** – تحويل النصوص العربية والإنجليزية والأرقام إلى رموز مورس والعكس.

## 🔒 الأمان
- واجهة تسجيل دخول بثلاثة حسابات مسبقة.
- فصل كامل بين واجهة المستخدم ونظام التشفير.

## 🎨 التقنيات المستخدمة
- HTML5
- CSS3 (تصميم زجاجي Glassmorphism، متجاوب مع جميع الشاشات)
- JavaScript (Vanilla - بدون مكتبات خارجية)

## 🚀 المميزات
- تبويبات منفصلة لكل نظام (البيجر / المورس)
- زر نسخ سريع للنتائج
- تصميم عصري بخلفيات متحركة وتأثيرات ضوئية
- دعم كامل للغة العربية (RTL)

## 📂 طريقة التشغيل
1. تحميل الملفات
2. فتح ملف `index.html` في أي متصفح حديث
3. استخدام بيانات الدخول المرفقة
